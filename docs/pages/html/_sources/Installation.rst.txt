Installation
################


Pypi
**********************

.. code-block:: console

    # Install from Pypi:
    pip install googletrends

    # Force update to latest version
    pip install -U googletrends


Github source
************************************

.. code-block:: console

    # Install directly from github
    pip install git+https://github.com/erdogant/googletrends


Uninstalling
################


Remove installation
**********************

Note that the removal of the environment will also remove the ``googletrends`` installation.

.. code-block:: console

    # Install from Pypi:
    pip uninstall googletrends
